Y=mx+b

Slope is the value of how inclined a line is in the XY plane. In the equation Y=mx+b, the slope is given by the value of m, which we can find from a graph by using rise (vertical distance from the origin), divided by run (horizontal distance from the origin).

The y-intercept is the point where the line intercepts the y-axis. In the equation Y=mx+b, the y-intercept is given by the value of b, which we can find from a graph by counting the distance of the line from 0 on the y-axis.
